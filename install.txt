#Выполните в консоли с правами суперпользователя:

Установка:
#cd /opt/
#tar -xvzf путь к архиву с биллингом fabfile.py

#apt-get update
apt-get install fabric postgresql-9.1 postgresql-contrib-9.1 postgresql-server-dev-9.1


#mkdir /opt/ebs

#su postgres
$createuser -P -s ebs
$createdb -O ebs ebs
$exit
$exit


Запустите утилиту установки биллинга с нуля. 
#fab deploy:demo.tar.gz 
При апгрейде с версии 1.4.1 нужно выполнить
#fab upgrade_14:demo.tar.gz


После установки откройте файл /opt/ebs/web/ebscab/settings_local.py и переопределите в нём значения из файла /opt/ebs/web/ebscab/settings.py.

После настройки веб-кабинета:

#/etc/init.d/apache2 reload

Внимание!!! Это версия не финальная версия 1.5. О всех найденных прблемах просьба писать на форуме http://forum.expertbilling.ru/ 

Имя и пароль по-умолчанию для входа в веб-кабинет admin/admin.
