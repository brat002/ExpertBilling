В модуле helpers имеются классы HeaderUtil, SplitterUtil которые предназначены для сохранения положения заголовков таблиц и сплиттера соответственно.
В них находятся методы для записи и получения состояния.


1 Заголовки таблиц.

В самом начале, в методе инициализации окна, должен быть вызван метод
getBinaryHeader(name)
который достанет (если он существует) заголовок, который был сохранён предыдущим окном. Заголовок сохраняется в переменную.

Затем определяется сигнал-событие  на изменение заголовка.
self.connect(tableHeader, QtCore.SIGNAL("sectionResized(int,int,int)"), self.saveHeader).

Перед первичным заполнением таблицы заголовок обнуляется(чтобы в него записались значения полученные функцией resizeColumnsToContents) и, после заполнения, если заголовок, который мы получили в самом начале не был пустым, устанавливаем первичный заголовок:
	setBinaryHeader(name, bhdr)
	getHeader(name, table).
	
В конце метода обновления таблицы необходимо восстанавливать заголовок:
	getHeader(name, table)
	
%-----
Стандартный метод который вызывается при срабатывании сигнала:
	def saveHeader(self, *args):
        if self.tableWidget.rowCount():
            HeaderUtil.saveHeader("class_frame_header", self.tableWidget)
 (если таблица пуста - значит не был сформирован автоматический заголовок по наполнению.)
