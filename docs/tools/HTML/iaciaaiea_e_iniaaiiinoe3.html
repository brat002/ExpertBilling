Назначение и особенности

Expert Billing System может работать с тремя типами карт: картами экспресс-оплаты, использующимися исключительно для пополнения баланса лицевого счета, картами доступа, которые при активации создают в системе нового абонента и HotSpot картами, предназначенными для активации доступа к услуге HotSpot на MikroTik  RouterOS.

 

Каждая карта характеризуется уникальным номером, серией, произвольным номиналом, уникальным PIN-кодом для активации, периодом возможной активации, статусом (активна/неактивна, продана/не продана)  и датой-временем активации.  Перед активацией любой тип карты должен быт передан на баланс дилеру.

 

Генерация номеров карт и уникальных случайных PIN-кодов происходит автоматически, управлять параметрами отдельных карт после генерации нельзя.

 

Каждую карту пользователь может активировать лишь единожды.
