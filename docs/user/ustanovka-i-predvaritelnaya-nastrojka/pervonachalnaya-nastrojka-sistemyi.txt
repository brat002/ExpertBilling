====================================
3.5 Первоначальная настройка системы
====================================


После завершения установки серверной части системы (кроме веб-кабинета
пользователя, о нем речь в отдельном разделе) рекомендуется произвести ряд
дополнительных первоначальных настроек.



По умолчанию в системе имеется два аккаунта администратора: admin с паролем
admin, предназначенный для работы с Expert Billing Admin, а также webcab,
предназначенный для взаимодействия пользователя через веб-кабинет с ядром
биллинга.



При смене логина или пароля пользователя webcab новые данные необходимо
указать в файле settings.py в полях RPC_USER и RPC_PASSWORD :

.. image:: _images/ebs_081.png

имя пользователя admin и пароль admin.
