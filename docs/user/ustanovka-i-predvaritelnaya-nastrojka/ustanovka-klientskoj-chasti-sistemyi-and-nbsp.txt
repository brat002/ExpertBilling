======================================
3.3 Установка клиентской части системы
======================================


Для работы с интерфейсом администратора (Expert Billing Admin) и интерфейсом
кассира, необходимо установить в систему нужные библиотеки.



Для этого необходимо запустить файл library_pack.exe из пакета дистрибутива и
распаковать его содержимое в папку C:\windows\system32 (или другую папку,
куда у вас установлена рабочая копия Windows). Допускается распаковка файлов
библиотек в папку с интерфейсом кассира и интерфейсом администратора.



Установка интерфейса администратора производится через файл EBSAdmin.exe

Установка интерфейса кассира производится через файл EBSCashKeeper.exe.
